# 🖊️ MeuPDF 50 - Assinador de PDFs Profissional

Um aplicativo web completo para assinar documentos PDF com recursos avançados de posicionamento, redimensionamento e rotação de assinaturas.

## ✨ Recursos

✅ **Carregamento de PDFs** - Suporta qualquer arquivo PDF
✅ **Desenho de Assinatura** - Interface intuitiva com SignaturePad
✅ **Posicionamento Preciso** - Arrastar, redimensionar e girar a assinatura
✅ **Multi-página** - Aplicar a mesma assinatura em todas as páginas ou apenas em uma
✅ **Responsivo** - Totalmente responsivo e otimizado para celular
✅ **Alerta de Orientação** - Pede para virar o celular em modo paisagem
✅ **Download Seguro** - Gera PDF assinado localmente (sem enviar para servidor)
✅ **Interface Moderna** - Design profissional e intuitivo

## 🚀 Como Usar

1. **Carregue um PDF** - Clique no botão "Selecionar PDF"
2. **Navegue pelas páginas** - Use os botões Anterior/Próxima
3. **Desenhe sua assinatura** - Clique em "Desenhar Assinatura"
4. **Posicione** - Arraste, redimensione e gire conforme necessário
5. **Aplicar a todas** - Marque a opção se quiser em todas as páginas
6. **Baixe** - Clique em "Baixar PDF Assinado"

## 📱 Mobile

Na primeira vez em celular, o app detecta se está em modo retrato e pede para virar para paisagem, oferecendo melhor experiência ao desenhar.

## 🛠️ Tecnologias

- **PDF.js** - Renderização de PDFs
- **SignaturePad** - Captura de assinatura
- **PDF-Lib** - Edição de PDFs
- **Vanilla JavaScript** - Sem dependências desnecessárias
- **CSS3** - Design responsivo

## 🔒 Privacidade

Todos os dados são processados **localmente no seu navegador**. Nenhuma informação é enviada para servidores.
